Demo > Preselecting values

You can preselect values even if they are not checked in the input model.

  • Result
  • View
  • Controller

Define your directive:

						
<div
	angular-multi-select
	input-model="input_data"
	output-model="output_data"

	dropdown-label="Click me!"
	node-label="<[ text ]>"
	leaf-label="<[ text ]>"
	preselect="value, a, value, e, id, 6"
></div>

Define your data models:

						
$scope.input_data = [
	{ text: 'A', value: 'a', id: 1},
	{ text: 'B', value: 'b', id: 2},
	{ text: 'C', value: 'c', id: 3},
	{ text: 'D', value: 'd', id: 4},
	{ text: 'E', value: 'e', id: 5},
	{ text: 'F', value: 'f', id: 6}
];

$scope.output_data = [];
