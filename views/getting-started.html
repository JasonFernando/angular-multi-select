Getting Started

Include the required files

First things first:

			
<!DOCTYPE html>
<html ng-app="myApp" id="myApp">
	<head>
		...
		<link rel="stylesheet" crossorigin="anonymous" href="angular-multi-select.css">
		...
		<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.5.0/angular.min.js"></script>
		<script src="https://cdnjs.cloudflare.com/ajax/libs/lokijs/1.3.16/lokijs.min.js"></script>
		<script src="angular-multi-select.js"></script>
		...
	<head>
It is very important to place crossorigin="anonymous" attribute in the <link> element, because that allows the directive to access it's own styles via documents.styleSheets.
Initiate your Angular app & load the module

In your main file 

			
var myApp = angular.module( "myApp", [ "angular-multi-select" ]);
In your view

Note Check Configuration & Options for a complete list of available options/attributes that can be used with the directive.

As an attribute.

					
<div
	angular-multi-select

	input-model="input_data"
	output-model="output_data"

	dropdown-label="Click me!"
	node-label="<[ text ]>"
	leaf-label="<[ text ]>"

	search-field="text"
</div>

... or as an element, but be careful with browser compatibility.

					
<angular-multi-select
	input-model="input_data"
	output-model="output_data"

	dropdown-label="Click me!"
	node-label="<[ text ]>"
	leaf-label="<[ text ]>"

	search-field="text"
>
</angular-multi-select>
In your controller

In your controller, define the input and output variables. This is the "ng-model" equivalent of this directive, so to speak.

Note The input model can be either an array of nested objects or a specially crafted array of objects (check Configuration & Options for a detailed explanation). 

			
$scope.input_data = [
	{
		text: 'A'
	},
	{
		text: 'B'
	},
	{
		text: 'C'
	},
	{
		text: 'D'
	}
];

$scope.output_data = [];

By now you should have a dropdown widget like the one you saw earlier on the front page.

Getting the selected items

Just iterate over the $scope.output_data in your controller. 

				
angular.forEach( $scope.output_data, function( value, key ) {
	/* do your stuff here */
});
