Events

Listening for events

AMS will broadcast events whenever something happens. The names of the events are:

  • ams_output_model_change
    • 					
$scope.$on('ams_output_model_change', function(event, args) {
	console.log(event, args);
});
					
				
    					
Object {name: "ams_output_model_change", targetScope: m, defaultPrevented: false, currentScope: b},
Object {name: "angular-multi-select-14609208330.14700627423064772"
    • name - the name you (or AMS) assigned to the directive


  • ams_input_model_change
    • 					
$scope.$on('ams_input_model_change', function(event, args) {
	console.log(event, args);
});
					
				
    					
Object {name: "ams_input_model_change", targetScope: m, defaultPrevented: false, currentScope: b},
Object {name: "angular-multi-select-14609208330.14700627423064772"
    • name - the name you (or AMS) assigned to the directive


  • ams_toggle_check_node
    • 					
$scope.$on('ams_toggle_check_node', function(event, args) {
	console.log(event, args);
});
					
				
    					
Object {name: "ams_toggle_check_node", targetScope: m, defaultPrevented: false, currentScope: b},
Object {name: "angular-multi-select-14609208330.14700627423064772", item: {"text":"A","value":"a","id":1,"open":false,"checked":1}}
    • name - the name you (or AMS) assigned to the directive
    • item - the item that caused the broadcast before the actual change
