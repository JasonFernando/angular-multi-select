Demo > Limited selection mode

The limited selection mode allows you to set the maximum amount of checked leafs. When that amount is reached, the oldest checked item will be unchecked.

This demo is limited to 2 checked leafs, and none is checked by default.

  • Result
  • View
  • Controller

Define your directive:

						
<div
	angular-multi-select
	input-model="input_data"
	output-model="output_data"

	dropdown-label="Click me!"
	node-label="<[ text ]>"
	leaf-label="<[ text ]>"

	max-checked-leafs="2"
></div>

Define your data models:

						
$scope.input_data = [
	{ text: 'A', value: 'a', id: 1},
	{ text: 'B', value: 'b', id: 2},
	{ text: 'C', value: 'c', id: 3},
	{ text: 'D', value: 'd', id: 4},
	{ text: 'E', value: 'e', id: 5},
	{ text: 'F', value: 'f', id: 6}
];

$scope.output_data = [];
