Shortcuts

You can easily control AMS with the built-in shortcuts. Note that different browsers trigger shortcuts in a different ways. Here is a short list of the most common browsers/OSes.

Browser Windows Linux Mac
IE Alt + accesskey
Chrome Alt + accesskey Alt + accesskey Control + Alt + accesskey
Firefox Alt + Shift + accesskey Alt + Shift + accesskey Control + Alt + accesskey
Safari Control + Alt + accesskey

Available access keys:

  • a - Check all
  • n - Check none
  • r - Reset
Keyboard navigation

In addition to the shortcuts, you can also use some keys to control AMS:

  • ESC - close an open AMS
  • key up - move up the selection in the tree of items
  • key down - move down the selection in the tree of items
  • key right - open the currently selected item in the tree of items
  • key left - close the currently selected item in the tree of items
  • Space - check/uncheck the currently selected item in tree of items
